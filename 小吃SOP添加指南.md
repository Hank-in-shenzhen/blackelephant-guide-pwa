# 小吃类SOP添加指南

## 框架已搭建完成，现在可以直接添加小吃SOP了！

### 添加步骤：
1. 打开 `data/guide-data.json` 文件
2. 找到 `===== 小吃类SOP模板开始 =====` 部分
3. 复制 `小吃模板示例` 整个对象
4. 粘贴到模板示例和 `checklist-open` 之间
5. 修改对应字段内容即可

### 字段说明：
```json
"小吃英文或拼音key": {
    "title": "🍟 【小吃名称】 制作SOP",          // 前面的图标可以根据品类更换：🍟炸物、🥟小食、🍚主食等
    "content": "<h3>原料配方</h3><p>原料1：XX克</p><p>原料2：XX克</p><h3>制作步骤</h3><ol><li>步骤1</li><li>步骤2</li></ol><h3>注意事项</h3><p>注意事项1</p><p>🎥 <a href=\"视频链接\" target=\"_blank\">点击查看视频</a></p>",
    "type": "food",                           // 固定为"food"，不要修改
    "keywords": ["关键词1", "关键词2", "关键词3"],  // 搜索关键词，方便用户查找
    "category": "炸物",                        // 分类：炸物/小食/主食/招牌小吃
    "taste": "咸香",                          // 口味：咸香/甜口/酸辣/原味/香辣等
    "makingTime": "快速",                     // 制作时间：快速/中等/慢
    "flavors": ["咸香", "酥脆"]                // 风味标签，用于风味筛选
}
```

### 分类说明：
- **炸物**：所有油炸类食品
- **小食**：小吃、凉菜、点心等
- **主食**：饭、面、粉等填饱肚子的食品
- **招牌小吃**：店里的特色、主推小吃

### 注意事项：
1. 每个小吃的key必须唯一，不能重复
2. JSON格式要正确，最后一个字段后面不要有逗号
3. 内容中的HTML标签要正确闭合
4. 添加完成后可以测试搜索和筛选功能是否正常
