## 🚀 更新项目的方式

### 修改数据
1. 编辑 `data/guide-data.json` 文件
2. 提交并推送：
   ```bash
   git add .
   git commit -m "更新数据"
   git push
   ```
3. Vercel会自动部署，1-2分钟后生效

### 修改代码
1. 编辑对应文件（index.html/css/style.css/js/app.js）
2. 提交并推送（同上）
3. Vercel自动部署
