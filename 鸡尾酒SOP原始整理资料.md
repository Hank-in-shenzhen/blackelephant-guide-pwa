#### **尼格罗尼调制流程**

1. **加冰入杯**：向rocks杯中加入足量普通冰块(非方冰)
    
2. **依次注酒**：按顺序加入原料
    
    - 首先倒入40ml金酒(最轻酒体优先)
        
    - 接着加入30ml金巴利
        
    - 最后倒入30ml红味美思
        
3. **搅拌混合**：用吧勺轻柔搅拌10-15秒，使酒体充分融合
    
4. **装饰完成**：
    
    - 取2片新鲜陈皮，挤净皮油至杯中
        
    - 将陈皮片投入酒中作为装饰
        

#### **三、关键技术要点**

- **冰块选择**：必须使用普通冰块而非方冰，确保适当融化速度以呈现风味变化
    
- **金酒适配**：不同金酒需搭配不同味美思(如Tanqueray金酒配Martini Rosso)
    
- **比例调整**：可通过增加金酒量(>40ml)淡化金巴利的苦味

### **🍹 鸡尾酒名称**：金汤力  
- **英文名称**：Gin Tonic  

### **📝 配方调制**

| 原料 | 规格/用量 | 作用 |
| :--- | :--- | :--- |
| **金酒（Gin）** | 45ml（文中提及“清酒的用量在45ml”，应为金酒笔误） | 基酒，提供核心风味 |
| **汤力水（Tonic Water）** | 适量（未明确具体毫升数） | 调和基酒，增加气泡感 |
| **青柠（Lime）** | 1颗（用于挤汁及装饰） | 提供酸味与香气 |
| **冰块** | 足量 | 降温并维持口感 |

### **👨‍🍳 调制步骤**
1. **冰杯准备**：将空杯预先冰镇（文中提及“先要把这个金汤力先要去冰杯”）。  
2. **添加冰块**：向冰杯中加入足量冰块（“凿冰放入杯子”）。  
3. **青柠处理**：  
   - 青柠皮朝下，向杯中挤入青柠汁（“青柠挤的时候青柠皮要朝下”）。  
   - 将挤汁后的青柠复原形状，丢入杯中（“复原这个青柠的形状，丢入杯中”）。  
4. **倒入金酒**：加入45ml金酒（“倒进金酒，清酒的用量在45ml”，此处“清酒”应为金酒）。  
5. **注入汤力水**：缓慢倒入汤力水（“随着汤力水的倒入，结晶的青柠会慢慢融化”）。  
6. **提拉混合**：做缓慢的提拉动作完成混合（“做一个缓慢的提拉动作就可以了”）。

Dry Martini（干马天尼）调制指南

### **🍸 鸡尾酒名称**

**Dry Martini**（干马天尼）

### **📝 配方调制**

| 材料 | 用量 | 备注 |
| :--- | :--- | :--- |
| **Dry Gin**（干金酒） | 8份 | 不可使用风味金酒 |
| **Dry Vermouth**（干味美思） | 2份 | 可根据"dry"程度调整（如extra dry可减少用量） |
| **Orange Bitters**（橙味苦精） | 2-3滴 | 提味关键 |
| **装饰** | - | 橄榄（必备）、柠檬皮（增香） |

### **👨‍🍳 调制步骤**
1. **准备工具**：Mixing Glass（调酒杯）、吧勺、滤冰器、Martini杯（预冰至-10℃左右）
2. **调酒杯加冰**：填满冰块（确保保温效果）
3. **添加材料**：依次加入橙味苦精（2-3滴）、干味美思（2份）、干金酒（8份）
4. **调和（Stir）**：手腕发力搅拌至有明显阻力感（凭经验判断温度达标）
5. **滤入冰杯**：快速过滤酒液至预冰Martini杯，避免混入空气
6. **装饰完成**：放入橄榄，柠檬皮擦杯口增香（可选将柠檬皮放入酒中）

### **🍹 鸡尾酒名称**

**金菲士**（Gin Fizz）- 经典夏季清爽型鸡尾酒

### **📝 配方调制**

| 材料 | 用量 | 备注 |
| :--- | :--- | :--- |
| **基酒** | 45ml | 金酒（视频中使用"醍醐达金酒"） |
| **柠檬汁** | 20ml | 新鲜榨取 |
| **糖浆** | 10ml | 用于平衡酸甜 |
| **苏打水** | 适量 | 碳酸饮料，最后添加 |
| **装饰**（可选） | 红柚片 | 创意改良用，非经典配方 |

### **👨‍🍳 制作步骤**

#### **一、前期准备**
1. **冰杯处理**：取克林杯加入摇冰，或放入冷冻柜冰镇；过滤融化水分后静置备用。

#### **二、核心调制**
1. **基酒添加**：倒入45ml金酒（可根据杯型调整比例）。
2. **酸甜平衡**：依次加入20ml柠檬汁、10ml糖浆，搅拌后尝味调整。
3. **混合降温**：加入冰杯中的冰水，再次搅拌混合。

#### **三、最终成型**
1. **注入苏打水**：沿冰块与杯壁缝隙倒入苏打水至九分满（避免直接冲在冰块上）。
2. **提拉融合**：用吧勺提拉冰块使上下液体混合，让气泡自然上涌（不需搅拌）。
3. **创意改良**（可选）：添加红柚片改变风味，成为个人化版本。

### **🍹 鸡尾酒名称**

**枯木逢春**（Cocktail Vol.35）

### **📝 配方调制**

| 材料 | 用量 | 功能 |
| :--- | :--- | :--- |
| **龙舌兰** | 50ml | 基酒 |
| **蜜瓜利口酒** | 25ml | 增添蜜瓜清香与森绿色调 |
| **芦荟汁** | 70ml | 提升甜感与口感丰富度 |
| **柠檬** | 半颗 | 提供酸度平衡 |
| **气泡水** | 3/1杯 | 增加清凉感，中和甜度 |

### **👨‍🍳 制作步骤**
1. **准备阶段**：雪克壶中放入5-6颗冰块
2. **混合基底**：依次加入龙舌兰50ml、蜜瓜利口25ml、芦荟汁70ml，挤入半颗柠檬汁
3. **摇匀酒液**：雪克壶盖盖，Shake 10秒
4. **装杯处理**：杯中加满冰块，倒入摇匀的酒液
