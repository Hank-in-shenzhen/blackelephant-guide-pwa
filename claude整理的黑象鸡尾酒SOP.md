## 1. 🍹 尼格罗尼 (Negroni)

**酒精感**：烈　　**风味**：微苦 · 草本 · 橙香

### 配方

| 原料       | 用量 |
| :--------- | :--- |
| 金酒       | 40ml |
| 金巴利     | 30ml |
| 红味美思   | 30ml |
| 冰块       | 足量 |
| 新鲜橙子皮 | 2片  |

### 步骤

1.  **加冰入杯**：向rocks杯中加入**400ml左右**普通冰块(非方冰)。
2.  **依次注酒**：按顺序加入原料。
    - 首先倒入**40ml**金酒(最轻酒体优先)。
    - 接着加入**30ml**金巴利。
    - 最后倒入**30ml**红味美思。
3.  **搅拌混合**：用吧勺轻柔搅拌10-15秒，使酒体充分融合。
4.  **装饰完成**：
    - 取2片新鲜橙子皮，挤净皮油至杯中。
    - 将橙片投入酒中作为装饰。

## 2. 🍹 金汤力 (Gin Tonic)

**酒精感**：中　　**风味**：清爽 · 草本 · 柑橘

### 配方

| 原料                  | 规格/用量 | 作用                 |
| :-------------------- | :-------- | :------------------- |
| 金酒（Gin）           | 45ml      | 基酒，提供核心风味   |
| 汤力水（Tonic Water） | 适量      | 调和基酒，增加气泡感 |
| 青柠（Lime）          | 1颗       | 提供酸味与香气       |
| 冰块                  | 足量      | 降温并维持口感       |

### 步骤

1.  **冰杯准备**：将空杯预先冰镇。
2.  **添加冰块**：向冰杯中加入**满杯**冰块（约200-250ml）。
3.  **青柠处理**：
    - 青柠皮朝下，向杯中挤入**约15ml**青柠汁。
    - 将挤汁后的青柠复原形状，丢入杯中。
4.  **倒入金酒**：加入**45ml**金酒。
5.  **注入汤力水**：缓慢倒入汤力水至**九分满**。
6.  **提拉混合**：做缓慢的提拉动作完成混合。

## 3. 🍹 莫吉托 (MOJITO)

**酒精感**：中　　**风味**：清爽 · 薄荷 · 柑橘

### 配方

| 材料     | 用量         | 备注                        |
| :------- | :----------- | :-------------------------- |
| 朗姆酒   | 60ml         | 示例使用Havana Club 3年陈酿 |
| 薄荷叶   | 适量（多量） | 用于基底和装饰              |
| 青柠     | 1整只        | 取汁使用                    |
| 蔗糖糖浆 | 4~5勺        | 根据口味调整                |
| 碎冰     | 足量         | 分两次添加                  |

### 步骤

1.  **基底处理**：杯中放入**约15片**薄荷叶和**1整只青柠**榨取的汁（约30ml），轻轻按压（避免压破薄荷叶茎）。
2.  **混合原料**：加入**足量**碎冰、**60ml**朗姆酒和**4~5勺**蔗糖糖浆，充分调和至糖浆完全融化。
3.  **二次加冰**：补充碎冰至杯口。
4.  **装饰完成**：顶部放置薄荷叶轻拍（震破薄膜释放香气），插入吸管。

## 4. 🍹 干马天尼 (Dry Martini)

**酒精感**：烈　　**风味**：干烈 · 草本 · 微咸

### 配方

| 材料                       | 用量  | 备注                                         |
| :------------------------- | :---- | :------------------------------------------- |
| 干金酒（Dry Gin）          | 8份   | 不可使用风味金酒                             |
| 干味美思（Dry Vermouth）   | 2份   | 可根据"dry"程度调整（如extra dry可减少用量） |
| 橙味苦精（Orange Bitters） | 2-3滴 | 提味关键                                     |
| 装饰                       | -     | 橄榄（必备）、柠檬皮（增香）                 |

### 步骤

1.  **准备工具**：Mixing Glass（调酒杯）、吧勺、滤冰器、Martini杯（预冰至-10℃左右）。
2.  **调酒杯加冰**：填满**满杯**冰块（约200-250ml，确保保温效果）。
3.  **添加材料**：依次加入橙味苦精（2-3滴）、干味美思（2份）、干金酒（8份）。
4.  **调和（Stir）**：手腕发力搅拌至有明显阻力感（凭经验判断温度达标）。
5.  **滤入冰杯**：快速过滤酒液至预冰Martini杯，避免混入空气。
6.  **装饰完成**：放入橄榄，柠檬皮擦杯口增香（可选将柠檬皮放入酒中）。

## 9. 🍹 古典鸡尾酒 (Old-fashioned)

**酒精感**：烈　　**风味**：烟熏 · 微苦 · 橙香

### 配方

| 材料类型 | 具体名称                          | 备注                               |
| :------- | :-------------------------------- | :--------------------------------- |
| 基酒     | Bourbon Whiskey（波本威士忌）     | 示例品牌：Elijah Craig Small Batch |
| 苦精     | Angostura Bitters（安高天娜苦精） | -                                  |
| 甜味剂   | 方糖                              | -                                  |
| 增香剂   | 陈皮                              | 用于释放陈皮油                     |
| 稀释剂   | 苏打水                            | 用于稀释糖分                       |
| 容器     | 古典杯                            | -                                  |
| 其他     | 冰块                              | 需8-9块                            |

### 步骤

1.  **基底制备**：在古典杯中放入**1块**方糖，滴入**2-3滴**Angostura Bitters苦精。
2.  **增香处理**：加入陈皮，碾压方糖的同时挤压陈皮释放陈皮油（内在香味）。
3.  **糖分稀释**：倒入**约20ml**苏打水，搅拌至方糖完全溶解。
4.  **降温稀释**：加入8-9块冰块，倒入**约45ml**Bourbon Whiskey，匀速搅拌至温度降低。
5.  **表层增香**：再次挤压新鲜陈皮，使陈皮油滴入酒液表面（外在香味）。
